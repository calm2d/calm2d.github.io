CALM - Canvas And Lisp Magic - Guide

Brief

This guide will lead you to write a simple application with CALM.

Download & Decompress

If you haven't downloaded CALM, go to CALM - Latest Releases to get your copy. Currently supported Operating Systems are: Linux, macOS and Windows.

After downloading, you will get an archive file compressed in 7-zip format and you need to decompress it.

We are not going to describe the steps to decompress a 7-zip archive, since you must be brilliant enough to do that.

After decompressing, you will get a directory named calm with the following contents in it:



`-- calm
    |-- calm # calm.exe on Windows
    |-- calm-bin # does not exist on Windows
    |-- core.lisp
    |-- gallery
    |   |-- fan
    |   |   |-- canvas.lisp
    |   |   `-- ...
    |   |-- meditator
    |   |   |-- canvas.lisp
    |   |   `-- ...
    |   `-- ...
    `-- ...

add the folder calm into your PATH variable.


export PATH="~/Downloads/calm/:$PATH"
cd ~/Downloads/calm/gallery/mondrian/
calm

First Drawing

Create a folder my-first-drawing, inside the folder, create a file canvas.lisp with the following code: 

(in-package #:calm)

(swank:create-server :port 4242)

(defun draw ()
  (c:move-to 120 200)
  (c:set-font-size 42)
  (c:set-source-rgb 0.047 0.21 0.51)
  (c:select-font-face "cursive" :normal :bold)
  (c:show-text "Oh, impermanent."))
Inside your shell, change to the folder, and execute calm: 

cd my-first-drawing
calm
You should have what you expected.
